# react-kits-cli

The CLI for starting dev server, bundling, and common utilities.

> Should be installed as `devDependencies`

## Install

```sh
yarn add -D react-kits
```

## Commands

```sh
react-kits init <my-app-name>

react-kits start

react-kits build

react-kits build-dll

react-kits lint

react-kits clean
```
